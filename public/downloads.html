Diario do Motorista - Arquivos disponiveis

Instrucoes de Instalacao

  1. Baixe o arquivo diario_chofer_v4.0.0.zip
  2. Descompacte o arquivo
  3. Copie .env.example para .env.local e preencha os valores
  4. Execute npm install
  5. Execute npx prisma db push para criar as tabelas
  6. Execute npm run dev para testar localmente
  7. Faça deploy no Vercel
